Sobre




Se você decidiu treinar para alguma modalidade, para obter uma boa performance, a planilha de treino é que vai te dar direcionamento para o caminho correto.

Mas para o que serve as tão faladas planilhas de treino?
Elas servem para desenvolver habilidades naquela pessoa respeitando as fases, para evitar lesões, evitar desgastes desnecessários, respeitar o tempo de descanso do corpo e construir uma evolução gradativa.

Se quiser fazer uma simulação, é só clicar em "Vamos Treinar ?"
